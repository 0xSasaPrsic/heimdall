# Heimdall

Validator node for Matic Network. It uses peppermint, customized [Tendermint](https://github.com/tendermint/tendermint).

### Install from source 

Make sure your have go1.11+ already installed

### Install 
```bash 
$ make install 
```  

### Run-heimdall 
```bash 
$ heimdalld start
```

### Run rest server

```bash 
$ heimdalld rest-server 
```
